Class: Downloader

Downloader(url, minFileSize, maxRetries, retryDelay, splits)

Download the file using multiple chunks in parallel to increase the download the speed The server must support accept-ranges header.

Constructor

new Downloader(url, minFileSize, maxRetries, retryDelay, splits)

Download the file using multiple chunks in parallel to increase the download the speed The server must support accept-ranges header. if file length is lesser than the minFileSize in bytes, the simple download will be used
Parameters:
Name Type Default Description
url string The URL to download from.
minFileSize number the minimum file length in bytes to initiate chunked download. Default value is 1 MB.
maxRetries number 9 the number of retries will be used in case of a network failure. Default value is 9
retryDelay number 3000 the number of milli seconds to wait in case of a netowrk failure. Default value is 3 seconds.
splits number 10 the number of chunks to be used at a same time. Default value is 10.
