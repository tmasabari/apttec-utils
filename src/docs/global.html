loadScript(url, callbackOnLoad, isDefer, isAsync)

Includes the script object to the head of the document. https://developer.mozilla.org/en-US/docs/Web/API/HTMLScriptElement
Parameters:
Name Type Description
url string the url to download the script
callbackOnLoad function | call a function after the script is loaded
isDefer boolean | If the async attribute is absent but the defer attribute is present, then the script is executed when the page has finished parsing.
isAsync boolean | If the async attribute is present, then the script will be executed asynchronously as soon as it downloads. The defer attribute may be specified with the async attribute, so legacy browsers that only support defer (and not async) fall back to the defer behavior instead of the default blocking behavior. If neither attribute is present, then the script is fetched and executed immediately, blocking further parsing of the page.
